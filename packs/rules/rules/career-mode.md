# Career Mode Rules

Active when working with the career skill.

## Hard Constraints
- Always check `data/applications.csv` before creating new applications (prevent duplicates)
- Reference `data/META_RESUME.md` as mandatory source of truth for all resume content
- Never fabricate accomplishments, metrics, or experience
- Never claim skills or certifications the user doesn't have

## Workflow Requirements
- Every application package includes: tailored resume + cover letter
- Match resume language to job posting keywords without being dishonest
- Include specific metrics and outcomes from actual experience
- Research the hiring manager and company before tailoring
